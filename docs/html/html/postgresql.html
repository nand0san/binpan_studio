handlers.postgresql.setup(symbol: str, tick_interval: str, postgresql_host: str, postgresql_user: str, postgresql_database: str, postgres_klines: bool, postgres_atomic_trades: bool, postgres_agg_trades: bool, postgresql_port: int = 5432) → tuple¶

Setups the connection to the PostgreSQL database.

Parameters:

  • symbol – A symbol like “BTCUSDT”

  • tick_interval – A tick interval like “1m”

  • postgresql_host – A host name or ip address

  • postgresql_user – A user name

  • postgresql_database – A database name

  • postgres_klines – A boolean to indicate if klines are requested

  • postgres_atomic_trades – A boolean to indicate if atomic trades are requested

  • postgres_agg_trades – A boolean to indicate if aggregate trades are requested

  • postgresql_port – A port number

Returns:

Connection and cursor to the database

handlers.postgresql.create_connection(user: str, enc_password: str, host: str, port: int, database: str) → Tuple[connection, cursor]¶

Creates a connection to the PostgreSQL database.

Parameters:

  • user – User name

  • enc_password – Encrypted password.

  • host – Host name or ip address

  • port – Port number

  • database – Name of the database

Returns:

Returns a connection and a cursor to the database

handlers.postgresql.data_type_from_table(table: str) → str¶

Gets data type from table names: “trade”, “aggTrade”, “depthX”, “depth”, “bookTicker” or “kline”. “orderbook”, “orderbook_value”.

Parameters:

table (str) – A stream name like “btcbusd_trade”

Return str:

Data type or none if not detected

handlers.postgresql.get_valid_table_list(cursor) → list¶

Obtains a list of valid tables from the database. Drops internal tables like “pg_stat_statements” or “pg_buffercache”.

Parameters:

cursor – A cursor to the database

Returns:

Returns a list of valid tables

handlers.postgresql.sanitize_table_name(table_name: str) → str¶

Sanitizes a table name. Replaces “@” with “_”. Adds a prefix if the name starts with a number.

Parameters:

table_name – The table name

Returns:

Sanitized table name like “t_1btcusd_trade” or “btcusd_kline_1m”

handlers.postgresql.get_data_and_parse(cursor, table: str, symbol: str, tick_interval: str, time_zone: str, start_time: int, end_time: int, data_type: str)¶

Gets data from a table in the database and parses it to a dataframe.

Parameters:

  • cursor – Database cursor

  • table – Table name

  • symbol – Symbol like “BTCUSDT”

  • tick_interval – Tick interval like “1m”

  • time_zone – Time zone like “Europe/Madrid”

  • start_time – Timestamp in milliseconds

  • end_time – Timestamp in milliseconds

  • data_type – Data type like “kline”, “trade”, “aggTrade”. Types are Binance websocket stream name types.

Returns:

A dataframe with the data and pertinent columns.

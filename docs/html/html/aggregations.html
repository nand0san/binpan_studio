This module can manage data aggregation.

This module can be imported:

from handlers import aggregations

Data Aggregation.

handlers.aggregations.ohlc_group(data: DataFrame, column_to_ohlc: str, group_column: str) → DataFrame

Creates OHLC columns for a column based on group by other column values.

Parameters

  • data (pd.Dataframe) – A dataframe with at least two columns.

  • column_to_ohlc (str) – Column to sparse values to OHLC columns in each group.

  • group_column (str) – This column will be the grouping key.

Returns

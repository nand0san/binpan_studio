Files Module

This module can manage files and API keys in an encrypted API keys and secrets file called secret.py.

Can be imported this way:

from handlers import files

Files and Folders

handlers.files.create_dir(my_path: str)

Create a folder at the designated path if it doesn’t exist. No error if it already exists

Parameters

my_path (str) – Path for the new directory.

handlers.files.save_dataframe_to_csv(filename: str, data: DataFrame, col_sep=',', index=False, timestamp=True) → None

Save a dataframe in a csv with a separator between columns. Each field of each column will be enclosed in double quotes.

Parameters

  • filename (str) – The file name.

  • data (pd.DataFrame) – The dataframe with date in columns.

  • col_sep (str) – Default is ‘,’

  • index (bool) – Keep index or not when saving. DEfault is False (drop index)

  • timestamp (bool) – Add a timestamp to the file name automatically. Default is true.

handlers.files.append_dataframe_to_csv(filename: str, data: DataFrame, col_sep: str = ',', index: bool = False, header=True) → None

Add lines to a csv with separator to choose, a dataframe. Each field of each column will be enclosed in quotes

Parameters

  • filename (str) – The file to use.

  • data (pd.DataFrame) – Data to append to the file.

  • col_sep (str) – Expected column separators. Default is “,”

  • index (bool) – Keeps index in file as first column/s.

  • header (bool) – Keeps header. Default is True.

Secrets Archive

To add encrypted data to secret.py file.

handlers.files.add_api_key(api_key_value: str) → None

Checks if exists in secret.py file and if not, then adds a line with the API key (not secret API key) encrypted value for working with the package.

Parameters

api_key_value (str) – API key

handlers.files.add_api_secret(api_secret_value: str) → None

Checks if exists in secret.py file and if not, then adds a line with the API secret encrypted value for working with the package.

Parameters

api_secret_value (str) – API key

handlers.files.add_any_key(key: str, key_name: str) → None

Checks if exists in a file and if not, then adds a line with the api key value for working with the package.

Parameters

  • key (str) – Any API key or secret to encrypt in secret.py file.

  • key_name (str) – Variable name to import it later.

Example to add telegram data for notifications handler:

from binpan import handlers

handlers.add_any_key(key="xxxxxxxxxx", key_name="encoded_chat_id")
handlers.add_any_key(key="xxxxxxxxxx", key_name="encoded_telegram_bot_id")
